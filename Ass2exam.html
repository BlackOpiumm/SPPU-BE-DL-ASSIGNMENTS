# Build model
model = keras.Sequential([
    layers.Flatten(input_shape=(28, 28)),
    layers.Dense(256, activation='relu'),
    layers.Dropout(0.2),
    layers.Dense(128, activation='relu'),
    layers.Dropout(0.2),
    layers.Dense(64, activation='relu'),
    layers.Dense(10, activation='softmax')
])
In[7]:
# Compile
model.compile(optimizer=keras.optimizers.SGD(learning_rate=0.01)
              , loss='sparse_categorical_crossentropy'
              , metrics=['accuracy'])

# #(a) Import the necessary packages

# TensorFlow and Keras are used to build and train the neural network.

# layers module lets us create network layers such as Dense (fully connected) and Flatten.

# NumPy handles mathematical operations and arrays.

# Matplotlib is used to display graphs (accuracy and loss curves).
#  Purpose: To bring all tools required for deep learning and plotting into the program.


## (b) Load the training and testing data (MNIST)

# The MNIST dataset contains grayscale images of handwritten digits (09).

# It has 60,000 training images and 10,000 testing images, each of size 2828 pixels.

# The data is normalized by dividing pixel values by 255 so that all values lie between 0 and 1  this helps the model learn faster.

# Some images are displayed using Matplotlib to visually confirm that the dataset has been loaded correctly.
#  Purpose: To prepare and understand the input data before giving it to the neural network.


##(c) Define the network architecture using Keras

# A Sequential model is created, which means layers are stacked one after another.

# The Flatten layer converts the 2828 image into a 1-D vector of 784 values.

# The Dense hidden layer with 128 neurons uses the ReLU (Rectified Linear Unit) activation function to introduce non-linearity and help the model learn complex patterns.

# The output layer has 10 neurons (for 10 digits) and uses the Softmax activation function to output probabilities for each class.
#  Purpose: To design a simple feedforward neural network that can classify digits.


# #(d) Train the model using SGD

# The model is compiled with:

# Optimizer: Stochastic Gradient Descent (SGD)  updates weights in small steps based on error.

# Loss Function: Sparse Categorical Cross-Entropy  measures how far the models predictions are from actual labels.

# Metrics: Accuracy  monitors how many predictions are correct.

# The model is trained for 10 epochs (10 passes over the dataset) and uses 20% of training data for validation to check performance on unseen data.
#  Purpose: To teach the neural network to correctly identify handwritten digits.

# #(e) Evaluate the network

# After training, the model is evaluated on the test dataset using model.evaluate().

# It calculates the test loss (error) and test accuracy (correct predictions).

# Test accuracy usually comes around 9697%, meaning the model correctly classifies most digits.
#  Purpose: To measure how well the trained model performs on completely new, unseen data.


# #(f) Plot the training loss and accuracy

# Accuracy and loss values are recorded during training in the history object.

# Two plots are made:

# Accuracy vs. Epochs  shows how model accuracy improves with training.

# Loss vs. Epochs  shows how training and validation loss decrease over time.

# These graphs help to detect overfitting or underfitting.
#  Purpose: To visualize the learning process and ensure the model is improving correctly.
